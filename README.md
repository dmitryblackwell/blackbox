## Install software

1. Install [nodejs](https://nodejs.org/en/)
1. Install [java 11](https://jdk.java.net/java-se-ri/11)

## Getting Started
```
git clone https://github.com/dmitryblackwell/blackbox.git
cd blackbox
npm install
npm start
./mvn spring-boot:run
```
